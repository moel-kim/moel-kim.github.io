## React FC를 사용하는 방법

React.FC\<P>를 React에서 사용하기 위해서는 아래와 같이 작성하면 됩니다.<br />
React.FC\<P>는 화살표 함수에서는 작동하지만 function 컴포넌트에서는 동작하지 않습니다

```tsx
import React from 'react'

interface IProps {
  content: string
}

const MyComponent: React.FC<IProps> = (props) => {
  return <div>{props.content}</div>
}
```

만약 function함수로 React 컴포넌트를 작성하려고 한다면 아래와 같이 작성할 수 있습니다

```tsx
import React from 'react'

interface IProps {
  content: string
}

function MyComponent(props: IProps): JSX.Element {
  return <div>{props.content}</div>
}
```

저는 Typescript 에러를 피하기 위해 JSX.Element를 컴포넌트의 return 타입으로 명시하는걸 선호합니다
