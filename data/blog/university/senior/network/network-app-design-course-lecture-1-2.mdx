## Preliminary

- network != computer network
  - road, railway, eletric, social 네트워크도 존재
- computer network != Internet
  - 다른 네트워크도 존재
- internet != Internet
  - "a internet" vs "the Internet"
  - internet(망을 연결한 망) vs intranet(중앙대 내부망, 삼성전자 내부망 등)
- the Internet을 배우려고 한다. 이는 컴퓨터 네트워킹의 기본적인 개념과 원리를 이해하기 위함이다.

## What's the Internet?

- network edge: 모든 종단 기기들(hosts = end systems)
- access network: wireless, wired link등 물리적인 매체들
- network core: 서로 연결된 패킷 스위치들(네트워크 라우터, 스위치 등등)
  - Network of Networks라고 불림.
  - 주로 해저나 눈에 보이지 않는 곳에 설치되어있음
- Internet: network of networks
  - internetworking of networks
  - interconnected ISPs
- protocols: control sending, receiving of messages
  - e.g. TCP, UDP, ICMP, ARP, RARP, IP, ...
- Internet standards(인터넷은 기준을 따르도록 설계되어있음)
  - RFC: Request for comments
    - RFC 791: Internet Protocol
    - RFC 793: Transmission Control Protocol
    - RFC 2616: Hypertext Transfer Protocol
    - ...
  - IETF: Internet Engineering Task Force
